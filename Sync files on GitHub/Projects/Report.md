# Final Report & Reflection

**Date of Submission:** 11 June 2026
**Weighting:** 40%

## Deliverables
1. **Main Group Report**: Formal documentation of all technical and non-technical work. Justify all design decisions and include test results.
2. **Professional Reflection Forms**: Must be prepared individually and added to the appendix (Must pass component!).

## Proposed Report Structure
- **Introduction**: Project brief and objectives.
- **Hardware Design**: Circuit diagrams, op-amp calculations, sensor selection (e.g., SS49E hall sensor).
- **Software Design**: State machine, WiFi communication, UART decoding, filtering algorithms.
- **Testing & Results**: Oscilloscope screenshots, motor diagnostic logs (e.g., test on 13/05).
- **Project Management**: Budget tracking, task allocation, timeline.
- **Conclusion & Future Improvements**.

*Note: One team member should act as the overall editor to ensure consistency in style and content.*
